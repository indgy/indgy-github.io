Letter spacing

Letter spacing supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

ls-[scale] (0-5)
{
 letter-spacing: [scale]rem;
}
-ls-[scale] (0-5)
{
 letter-spacing: -[scale]rem;
}

Usage

Letter spacing

ls-[0-5]

Negative letter spacing

-ls-[0-5]
