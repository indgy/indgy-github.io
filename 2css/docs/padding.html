Padding

Padding supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

pa-[scale] (0-19)
{
 padding: [scale]rem;
}
ph-[scale] (0-19)
{
 padding-left: [scale]rem;
 padding-right: [scale]rem;
}
pv-[scale] (0-19)
{
 padding-top: [scale]rem;
 padding-bottom: [scale]rem;
}
p[position]-[scale] (top, right, bottom, left), (0-19)
{
 padding-[position]: [scale]rem;
}
