Borders

Borders use the pixel scale (0 to 9) for the border width, border radius uses the rem scale (0-19), border colour uses the alpha scale (01-100) Only black and white border colours are provided as rgba() colours, they will lighten or darken the box outline.

Synopsis

br-[scale] (0-9)
{
 border-style: solid;
 border-width: [scale]px;
}
br-[position]-[scale] (top, right, bottom, left), (0-9)
{
 border-[position]-style: solid;
 border-[position]-width: [scale]px;
}
rd-[radius] (0-19)
{
 border-radius: [radius]rem;
}
rd-[position]-[radius] (top, right, bottom, left), (0-19)
{
 border-[position]-radius: [radius]rem;
}
br-[colour]-[scale] (black, white), (01-100)
{
 border-color: [color];
}

Usage

Border

br-[0-9]

Border, positional

br-t-[0-9]
br-r-[0-9]
br-b-[0-9]
br-l-[0-9]

Border radius

rd-[0-19]

Border radius, positional

rd-t-[0-19]
rd-r-[0-19]
rd-b-[0-19]

Border colour

br-black-[01-100]
br-white-[01-100]

Examples

Use position to target a border side

br-l-8 br-black-50

Combine sides to create unique effects

br-1 br-b-4 br-black-20
