Height

Height offers three scales, percentages (h), rems (h) and viewport height (hv). Additionally min and max heights are supported for all three scales.

Height supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

h-[scale] (0-100)
{
 height: [scale]%;
}
mx-h-[scale] (0-100)
{
 max-height: [scale]%;
}
mn-h-[scale] (0-100)
{
 min-height: [scale]%;
}
h-[scale] (0-19)
{
 height: [scale]rem;
}
mx-h-[scale] (0-19)
{
 max-height: [scale]rem;
}
mn-h-[scale] (0-19)
{
 min-height: [scale]rem;
}
hv-[scale] (0-100)
{
 height: [scale]vh;
}
mx-hv-[scale] (0-100)
{
 max-height: [scale]vh;
}
mn-hv-[scale] (0-100)
{
 min-height: [scale]vh;
}

Usage

Height as percentage

h-[0-100]

Max-height as percentage

mx-h-[0-100]

Min-height as percentage

mn-h-[0-100]

Height in rems

h-[0-19]

Max-height in rems

mx-h-[0-19]

Min-height in rems

mn-h-[0-19]

Max-height in viewport units

mx-hv-[0-100]

mn-hv-[0-100]
