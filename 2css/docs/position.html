Position

Position uses the rhythm scale (rems)

Position supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

[position] (absolute, fixed, relative, static)
{
 position: [position];
}
t-[scale] (0-19)
{
 top: [scale]rem;
}
-t-[scale] (0-19)
{
 top: -[scale]rem;
}
r-[scale] (0-19)
{
 right: [scale]rem;
}
-r-[scale] (0-19)
{
 right: -[scale]rem;
}
b-[scale] (0-19)
{
 bottom: [scale]rem;
}
-b-[scale] (0-19)
{
 bottom: -[scale]rem;
}
l-[scale] (0-19)
{
 left: [scale]rem;
}

{
 left: -[scale]rem;
}
