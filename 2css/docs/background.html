Background

Background supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

bg-[attach] (fixed, local, scroll)
{
 background-attachment: [attach];
}
bg-[position] (center center, center top, top right, center right, bottom right, center bottom, bottom left, center left, top left)
{
 background-position: [position];
}
bg-[repeat] (repeat, repeat-x, repeat-y, no-repeat, no-repeat)
{
 background-repeat: [repeat];
}
bg-[size] (auto, cover, contain)
{
 background-size: [size];
}
bg-[color]-[scale] (01-100)
{
 background-color: [color];
}
bg-[color]-[scale] (0-9)
{
 background-color: [color];
}
-bg-[color]-[scale] (0-9)
{
 background-color: [color];
}
