Lists

Lists supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

ls-[styles] (inline, none)
{
 list-style: [styles];
}
lst-[styles] (circle, disc)
{
 list-style-type: [styles];
}
lsp-[styles] (inside, outside)
{
 list-style-position: [styles];
}

Usage

List

ls-i
ls-n

List style

lst-c
lst-d

List position

lsp-i
lsp-o
