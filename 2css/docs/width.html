Width

Width offers three scales, percentages (h), rems (h) and viewport width (hv). Additionally min and max widths are supported for all three scales.

Width supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

w-[scale] (0-100)
{
 width: [scale]%;
}
mx-w-[scale] (0-100)
{
 max-width: [scale]%;
}
mn-w-[scale] (0-100)
{
 min-width: [scale]%;
}
w-[scale] (0-19)
{
 width: [scale]rem;
}
mx-w-[scale] (0-19)
{
 max-width: [scale]rem;
}
mn-w-[scale] (0-19)
{
 min-width: [scale]rem;
}
wv-[scale] (0-100)
{
 width: [scale]vh;
}
mx-wv-[scale] (0-100)
{
 max-width: [scale]vh;
}
mn-wv-[scale] (0-100)
{
 min-width: [scale]vh;
}

Usage

Width as percentage

w-[0-100]

Max-width as percentage

mx-w-[0-100]

Min-width as percentage

mn-w-[0-100]

Width in rems

w-[0-19]

Max-width in rems

mx-w-[0-19]

Min-width in rems

mn-w-[0-19]

Width in viewport units

wv-[0-100]

Max-width in viewport units

mx-wv-[0-100]

Min-width in viewport units

mn-wv-[0-100]
