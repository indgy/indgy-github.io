Line height

Line height supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

lh-[scale] (0-5)
{
 line-height: [scale]rem;
}
-lh-[scale] (0-5)
{
 line-height: [scale]rem;
}

Usage

Line height

lh-[0-5]

Negative line height

-lh-[0-5]
