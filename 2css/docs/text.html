Text

Text supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

t[align] (left, right, center, justify)
{
 text-align: [align];
}
td-[decoration] (line-through, underline, none)
{
 text-decoration: [decoration];
}
tt-[transform] (capitalize, lowercase, uppercase, none)
{
 text-transform: [transform];
}
tt-sc
{
 font-variant: small-caps;
}
