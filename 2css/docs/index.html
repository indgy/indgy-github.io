2css

A modern complement to atomic CSS frameworks like Tachyons and TailwindCSS.

The aim

  1. Start with a css reset for cross browser consistency, a number of good ones are bundled.
  2. Add a little no-css so your plain html looks as good as can be without adding any extra styling - less code, less work, less size.
  3. Add consistent utility classes using scales that are easy to remember.
    1. Class names are the two letter mnemonics of the css properties wherever possible.
    2. Scales run continuously from 0 to 9 or 0 to 19 for pixels and rems and 0-100 for percentage or viewport units.
  4. Generate plain .scss files for you to incorporate into your project.

Why is it named '2css'?

2css is the result of an experiment to try reproduce every css rule in two letters.

Each class should read exactly as the css rules would in your mind, without all the other letters.

2css is mainly influenced by Tachyons however it does copy some great ideas from TailwindCSS, namely the media query prefixes (xs:, sm:, md:, lg:, xl:) and the :hover effects on colours.

Plus we add a new one - literal '%' units when working with widths and heights, these represent literal percentage units.

Some examples

fl
controls the 'float' property 
fl-l
floats left 
fl-r
floats right 
pr-4
adds 4 units of right padding 
mt-8
adds 8 units of top margin 
fs-4
sets font size to 4 units 
lh-2
increases line height by 2 units 
-lh-2
decreases line height by 2 units 
w-50%
sets width to 50% 
mx-w-50%
sets max-width to 50% 
mx-vw-50
sets max-width to 50vh 
mx-w-80% md:mx-w-60% lg:mx-w-50%
max-width is responsive across different screen widths

NB: All distances and font sizes depend on your scale.
