Margin

Margin supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

ma-[scale] (0-19)
{
 margin: [scale]rem;
}
-ma-[scale] (0-19)
{
 margin: -[scale]rem;
}
mh-[scale] (0-19)
{
 margin-left: [scale]rem;
 margin-right: [scale]rem;
}
mv-[scale] (0-19)
{
 margin-top: [scale]rem;
 margin-bottom: [scale]rem;
}
m[position]-[scale] (top, right, bottom, left), (0-19)
{
 margin-[position]: [scale]rem;
}
-m[position]-[scale] (top, right, bottom, left), (0-19)
{
 margin-[position]: -[scale]rem;
}
