Fonts

Font sizes use the type scale, colours use the solid color (0-9) or the opacity (01-100) scale.

Fonts supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

fc-[color]-[scale] (black, white, brown, blue, purple, pink, red, orange, yellow, green), (0-9)
{
 color: [color];
}
-fc-[color]-[scale] (black, white, brown, blue, purple, pink, red, orange, yellow, green), (0-9)
{
 color: [color];
}
fc-[color]-[scale] (black, white, brown, blue, purple, pink, red, orange, yellow, green), (01-100)
{
 color: [color];
}
ff-[name] (system-ui, -apple-system, Blinkmacsystemfont, "segoe Ui", roboto, "Helvetica neue", Arial, "noto sans", sans-serif, "Apple Color emoji", "segoe Ui emoji", "segoe Ui symbol", "noto Color emoji";, Palatino, Georgia, "Lucida Bright", "Book Antiqua", "Times new roman", Times, serif;, menlo, monaco, Consolas, "Liberation mono", "Courier new", monospace;)
{
 font-family: [name];
}
fs-[scale] (0-9)
{
 font-size: [scale]rem;
}
-fs-[scale] (3-1)
{
 font-size: [scale]rem;
}
fs-[style] (italic, normal)
{
 font-style: [style];
}
fw-[weight] (100-900)
{
 font-weight: [weight];
}
