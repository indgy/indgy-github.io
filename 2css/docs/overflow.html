Overflow

Overflow supports media query modifiers (xs:, sm:, md:, lg:, xl:)

Synopsis

of-[values] (auto, hidden, scroll, visible)
{
 overflow: [values];
}
ofx-[values] (auto, hidden, scroll, visible)
{
 overflow-x: [values];
}
ofy-[values] (auto, hidden, scroll, visible)
{
 overflow-y: [values];
}
